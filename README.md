# 서적 쇼핑몰 서비스
장편소설, 중편소설, 단편소설로 카테고리화한 소설 전문 쇼핑몰.

### 1. 기능
- 회원관리
  - 회원가입/로그인
  - 개인정보관리
  - 주문관리
  - 장바구니
- 도서 검색 및 분류
  - 카테고리 검색 필터링
  - 도서 상세정보 제공
- 결제 시스템
  - 카드결제 외부 api 도입하여 결제 진행

### 2. 팀 구성
- 윤주연 (CTO)
- 박조은
- 박영욱
- 허민주
- 이셀라
- 김동현

### 3. 개발 환경
- React 19
- JavaScript
- React Router 7
- styled-components
- Vercel
